# Golden Agent 🔸

**Meet the AI agent that gets better the more you use it.** Unlike other agents, this one actually learns as it goes — it picks up new skills on the fly, keeps getting sharper over time, reminds itself to save important things, looks back at past conversations, and slowly builds a real picture of who you are.

## Quick Install

For Windows
```bash
irm https://raw.githubusercontent.com/mochiyaki/gold/main/scripts/install.ps1 | iex
```

For macOS, Linux
```bash
curl -fsSL https://raw.githubusercontent.com/mochiyaki/gold/main/scripts/install.sh | bash
```

After installation, type
```bash
gold
```

For simple or fast start, execute
```bash
gold --lite
```
